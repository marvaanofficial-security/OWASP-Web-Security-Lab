# OWASP Top 10 Overview

This lab focuses on understanding and practicing vulnerabilities from the OWASP Top 10 in a safe environment.

Topics covered
- Cryptographic Failures
- Injection
- Security Misconfiguration
- Vulnerable Components
- Identification and Authentication Failures
- Security Logging and Monitoring Failures
  

For each topic, testing was performed only within the OWASP Juice Shop lab environment.
