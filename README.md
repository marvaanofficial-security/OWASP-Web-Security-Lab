# 🛡️ Web Application Security Lab

## Overview

This project is a hands-on Web Application Security Lab built to practice identifying, exploiting, and documenting common web application vulnerabilities using industry-standard tools. The lab is based on the OWASP Top 10 and simulates real-world penetration testing scenarios.

## Objectives

- Learn web application penetration testing
- Understand OWASP Top 10 vulnerabilities
- Practice using Burp Suite
- Analyze HTTP requests and responses
- Document security findings professionally

## Technologies Used

- Kali Linux
- Docker
- OWASP Juice Shop
- Burp Suite Community Edition
- Mozilla Firefox

## Lab Components

- Burp Suite Proxy
- HTTP Request Interception
- Repeater
- Intruder
- OWASP Juice Shop
- Docker Environment

## Vulnerabilities Practiced

- SQL Injection (SQLi)
- Cross-Site Scripting (XSS)
- Broken Authentication
- Broken Access Control
